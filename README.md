1. Запуск приложения:
   - открыть терминал;
   - перейти в папку с программой (/project_1)
   - прописать команду в терминале "python main.py" (если не сработает, то "python3 main.py")
2. Функционал:
   Команды:
   - add - добавить задачу; (чтобы пропустить создание описания задачи, нажмите Enter)
   - disc — отобразить описание задачи;
   - list - показать текущие задачи;
   - done - отметить задачу выполненной;
   - delete - удалить задачу;
   - help - показать список команд;
   - exit - завершить работу.
  
*файл README будет изменяться по мере доработки программы.
